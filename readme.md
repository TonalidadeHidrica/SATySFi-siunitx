This is a comprehensive (SI) units package of SATySFi.


# install

## Satyrographos

```
git clone https://github.com/puripuri2100/SATySFi-siunitx.git

cd SATySFi-siunitx

opam pin add -v "file://$PWD"

satyrographos install
```
